* ":=" VS "="  
":="在赋值的同时，会将赋值语句中所有的变量就地展开，也就是说，A:=$(B)后，B的值的改变不再影响A。  
"=" 表示变量的值将会是整个makefile中最后被指定的值。

* ?=
如果没有被赋值过就赋予等号后面的值。

* 内置函数

| 函数 | 作用 | 用法 |
| ------ | ------ | ------ |
| ifneq | 比较是否不同 | ifneq (arg1,arg2） |
| sed | 替换 | sed 's/旧字符串/新字符串/g'  |
| patsubst | 模式替换 | $(patsubst <pattern>,<replacement>,<text> ) |
